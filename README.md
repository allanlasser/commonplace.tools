# backend.tools
Decentralized Tool Share

## Frontend

The frontend uses SvelteKit. It's fun and nice!

### Development

```sh
cd frontend
npm install
# if you're running locally:
npm run dev -- --open
# if you're using Codespaces:
npm run dev -- --host
```

If you're using Codespaces, using `--host` exposes the port to the network.
